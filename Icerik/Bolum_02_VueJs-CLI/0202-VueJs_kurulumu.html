VueJs kullanmaya başlama

VueJs'i ile kodlama yapabilmek için iki yöntemden birini kullanabilirsiniz.

  1. CDN ile kullanım

    2. CDN (Content Delivery Network/İçerik Dağıtım Ağı) ile değişik dosya içeriklerine online olarak ulaşabilirsiniz. Bazı JavaScript dosyalarına da web sayfalarınızdan içerik dağıtım ağına (CDN) referans vermek suretiyle ulaşıp oradaki halini kullanabilirsiniz. Bunun için sayafanızın başına (head elementi içine) aşağıdaki kod satırlarından birini eklemeniz yeterli olacaktır.

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script src="https://unpkg.com/vue@next"></script>

  2. CLI kurulumu ile kullanım

    3. VueJs CLI ile çalışmaya başlamak için önce sisteminize kurmanız gerekmektedir.

      Kurulum için

    1. VueJs CLI'yi bilgisayara kurmak için npm (node package manager) kullanıyoruz.
    2. nmp nodejs ile çalıştığı için bilgisayarınızda node.js kurulu olması gerekir
    3. nodejs kurulu olup olmadığını anlamak için vscode üzerindeki terminal ekranından veya cmd penceresinden node -v komutu vermeniz yeterlidir.
    4. Bu moutu verince node versiyonunu veriyorsa sorun yok vermiyorsa/hata ile karşılaşıyorsanız nodejs'yi indirip kurmanız gerekir.
    5. https://nodejs.org/en/download/ linkinde kendinize uygun olanını indirip kurmanız gerekir.
    6. Kurulum normal tamamlandıktan sonra vscode u yeniden başlatıp terminal penceresine veya windows'un cmd penceresine node -v komutu ile kontrol edebilirsiniz.
    7. node.js ile birlikte bilgisayarınıza npm de (node package manager) yüklenmiş olacaktır.
    8. node -v komutundan olumlu sonuç alıyorsanız, npm -v ile npm versiyonunu kontrol edin.
    9. herhangi bir sıkıntı yok ise npm install vue@next ile vue@next kurulumunu yapın
    10. npm install -g @vue/cli komutu ile vue/cli yükleyin
    11. vue create my-new-app ile uygulamamızı oluşturuyoruz.
    12. vue create my-new-app ile uygulamamızı oluşturken ...AppData\Roaming\npm\vue.ps1 cannot be loaded because running scripts is disabled on this system hatası alırsanız power shell'de aşağıdaki komutu çalıştırabilirsiniz.
    13. Set-ExecutionPolicy RemoteSigned bu komut ile ilgili detaya bu linkten ulaşabilirsiniz.
    14. Proje içerisine girip:
    15. vue upgrade - next komutunu çalıştırıyoruz.
    16. npm run dev komutu ile projemizi çalıştırıyoruz.
